Taxi Trip Insights


The New York City Taxi Commission collects data for millions of Taxi trips every month

Our app seeks to communicate some of these trends in these services from 2016 to the present

The following is a useful and reliable product of the shift towards Big Data

Using basic database querying and a modern web framework we aim to present this information in an effective and simple way
